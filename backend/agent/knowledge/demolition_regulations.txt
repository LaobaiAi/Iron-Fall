本规范基于《钢结构工程施工质量验收规范》GB50205 和《建筑拆除工程安全技术规范》JGJ147 编制。
"""

REGULATIONS = """
# 第一章 总则

## 1.1 编制依据
- 《钢结构工程施工质量验收规范》GB50205
- 《建筑拆除工程安全技术规范》JGJ147
- 《钢结构设计标准》GB50017
- 《建筑施工高处作业安全技术规范》JGJ80

## 1.2 适用范围
本规范适用于工业与民用建筑钢结构拆除工程的施工安全管理。

# 第二章 基本规定

## 2.1 一般要求
1. 钢结构拆除前，应编制拆除施工方案，经审批后方可实施。
2. 拆除施工前，应对被拆除钢结构进行全面勘察，了解结构形式、连接方式、受力状态。
3. 拆除作业人员必须持有相应的资格证书。

## 2.2 拆除顺序原则
1. **先次要后主要**: 先拆除非承重构件和附属设施，后拆除承重构件。
2. **自上而下**: 对于多层建筑，应从上向下逐层拆除。
3. **对称均衡**: 避免不对称拆除导致结构产生扭转失稳。
4. **分段卸载**: 大型构件应采取分段卸载措施，控制变形。

# 第三章 构件拆除要求

## 3.1 柱类构件
### 3.1.1 底层柱
**严禁拆除底层主要承重柱**，如因特殊情况需要拆除，必须满足以下条件：
- 已设置可靠的临时支撑替换体系
- 经过具有相应资质的设计单位验算确认
- 制定详细的安全监控方案

### 3.1.2 非底层柱
非底层柱拆除时应：
- 确认上部结构有可靠的传力路径
- 避免同时拆除相邻层的同位置柱
- 必要时设置侧向支撑

## 3.2 梁类构件
1. 主梁拆除前应验算次梁的承载能力。
2. 悬臂梁严禁拆除悬挑端。
3. 连续梁拆除时应考虑连续跨的影响。

## 3.3 支撑系统
1. 支撑系统是结构整体稳定的关键，严禁提前拆除。
2. 水平支撑和垂直支撑应配合拆除。
3. 拆除支撑前必须确认荷载已有效传递。

# 第四章 稳定性验算

## 4.1 验算要求
1. 拆除方案实施前，必须进行结构稳定性验算。
2. 验算应包括：承载能力、变形、整体稳定性。
3. 验算结果应满足设计要求的安全系数。

## 4.2 拆除验算工况
1. **单步拆除**: 拆除单个构件后的结构状态
2. **累积拆除**: 多步拆除后的结构累积效应
3. **偶然荷载**: 考虑风荷载、地震作用的不利组合

## 4.3 稳定性判据
- 最大位移不超过结构高度的 1/500
- 构件应力不超过设计强度的 85%
- 结构稳定系数大于 1.5

# 第五章 风险控制

## 5.1 高风险作业识别
以下情况视为高风险作业：
1. 拆除底层或地下室柱
2. 拆除带支撑的构件
3. 同时拆除超过 3 个主构件
4. 结构存在损伤或腐蚀

## 5.2 安全措施
1. 设置安全警戒区域
2. 配备专职安全监护人员
3. 实施全过程变形监测
4. 制定应急预案

## 5.3 监测要求
1. 关键构件应安装应力传感器
2. 重要节点应设置位移监测点
3. 发现异常应立即停止作业

# 第六章 应急预案

## 6.1 预警指标
- 监测位移超过允许值的 70% 时发出预警
- 监测位移超过允许值时应立即停止作业

## 6.2 应急响应
1. 立即启动应急预案
2. 组织人员有序撤离
3. 采取临时支撑措施
4. 评估结构状态

# 附录 A 拆除方案审查要点

1. 结构分析报告完整性
2. 验算结果安全性
3. 拆除顺序合理性
4. 安全措施可行性
5. 应急预案有效性

# 附录 B 常用构件识别

## 柱 (Column)
- 连接楼板和基础的竖向构件
- 主要承受轴向压力
- 是结构的垂直承重体系

## 梁 (Beam)
- 连接两根柱的水平构件
- 主要承受弯矩和剪力
- 传递楼面荷载至柱

## 支撑 (Brace)
- 斜向连接构件
- 主要承受轴向力
- 提供结构侧向稳定性
"""

if __name__ == "__main__":
    print(REGULATIONS)
